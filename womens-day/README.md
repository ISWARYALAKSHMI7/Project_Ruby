## Color Scheme

The website uses a sophisticated gradient palette:
- Pink (#ec4899) - Primary brand color
- Purple (#a855f7) - Secondary accent
- Rose (#f43f5e) - Tertiary accent
- Red (#ef4444) - Logo color

## Logo

The website expects a logo file at `public/logo.jpg` in red color. If the image is not found, it displays a beautiful fallback with the Ruby Wellness Center name.
